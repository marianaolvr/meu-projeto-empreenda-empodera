  • App EMP

    • O App EMP foi criado para facilitar a interação entre contratante x contratada.
    Um app intuitivo e fácil de usar, você pode se cadastrar para contratar alguém, para ofertar seu serviço, ou para ambos os casos.
    Para ofertar seus serviços, basta se cadastrar de acordo com a categoria do seu ramo de atuação e seguir as instruções do App.
    Para contratar um serviço, basta selecionar a categoria desejada e seguir as instruções do app.
